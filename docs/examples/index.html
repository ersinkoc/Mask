Basic Examples

Start with these fundamental examples:

01. Basic Email Masking

Learn the basics of email masking with different strategies.

mask.email('john.doe@example.com');
// → j***n.***@e***e.com
View Example →

02. Custom Strategies

Explore all 5 masking strategies with practical examples.

mask.email('test@example.com', {
  strategy: 'first:2'
});
// → te**@e***e.com
View Example →

03. Phone Masking

Handle international phone formats with country codes.

mask.phone('+1-555-123-4567');
// → +1-***-***-4567
View Example →

04. Credit Card Masking

Mask credit cards with Luhn validation and formatting.

mask.card('4532015112830366');
// → **** **** **** 0366
View Example →

05. Custom Masking Characters

Use different characters like •, █, # for masking.

mask.email('test@example.com', {
  char: '•'
});
// → t•••t@e•••.c••
View Example →

06. Multiple Data Types

Work with email, phone, card, IBAN, and IP addresses.

// Email, phone, card, IBAN, IP
mask.email('test@example.com');
mask.phone('+1-555-123-4567');

Advanced Examples

Learn advanced techniques and patterns:

01. Object Masking

Mask entire objects with selective field protection.

mask.mask(user, {
  fields: {
    email: { strategy: 'middle' },
    phone: { strategy: 'last:4' }
  }
});
View Example →

02. Formatting Options

Use display, compact, and log formats effectively.

// Display, Compact, Log
mask.card('4532015112830366', {
  format: 'display'
});
// → **** **** **** 0366

03. Logging Integration

Integrate masking with logging systems.

logger.info('User login', {
  email: 'user@example.com'
});
// → [INFO] User login [REDACTED:email]

04. Database Integration

Store masked data in databases safely.

Real-World Examples

Production-ready examples from real applications:

01. User Database

Complete user database example with masking.

class UserDatabase {
  addUser(user) {
    const masked = mask.mask(user, {
      fields: { email: { strategy: 'middle' } }
    });
    // Store masked data
  }
}

02. Logging Middleware

Express-style middleware for automatic request logging.

class LoggingMiddleware {
