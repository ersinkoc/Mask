Quick Start

Installation

npm / yarn / pnpm
npm install @oxog/mask
# or
yarn add @oxog/mask
# or
pnpm add @oxog/mask

Basic Usage

TypeScript / JavaScript
import { createMask } from '@oxog/mask';

const mask = createMask();
mask.use(require('@oxog/mask/plugins/core/email').email());
mask.use(require('@oxog/mask/plugins/core/phone').phone());
mask.use(require('@oxog/mask/plugins/core/card').card());

// Mask an email
mask.email('john.doe@example.com');
// ‚Üí j***n.***@e***e.com

// Mask with custom strategy
mask.email('john.doe@example.com', { strategy: 'full' });
// ‚Üí ***************@*********.***

// Mask a credit card
mask.card('4532015112830366');
// ‚Üí **** **** **** 0366

Masking Strategies

Choose from 5 built-in strategies to control how data is masked:

Strategy Description Example Result
full Mask entire value mask('ABCDEFGHIJ', { strategy: 'full' }) **********
middle Mask middle portion (default) mask('ABCDEFGHIJ', { strategy: 'middle' }) A********J
first:N Keep first N characters mask('ABCDEFGHIJ', { strategy: 'first:3' }) ABC*******
last:N Keep last N characters mask('ABCDEFGHIJ', { strategy: 'last:4' }) ******GHIJ
partial:N Mask partial percentage mask('ABCDEFGHIJ', { strategy: 'partial:0.5' }) ABCDE*****

Architecture

Micro-Kernel Design

@oxog/mask uses a micro-kernel architecture with a central orchestrator that manages plugin registration and execution. This design provides:

  • Extensibility: Add new masking types via plugins
  • Flexibility: Combine different strategies and formats
  • Maintainability: Each plugin is self-contained
  • Performance: Zero runtime dependencies

Plugin System

The library includes core plugins and optional plugins:

Core Plugins

  • email - Email masking
  • phone - Phone masking
  • card - Credit card masking

Optional Plugins

  • iban - IBAN masking
  • ip - IP address masking
  • jwt - JWT token masking
  • ssn - SSN masking
  • url - URL masking

Custom Plugins

Create your own plugins by implementing the plugin interface:

{
  name: 'custom',
  version: '1.0.0',
  install(kernel) {
    kernel.registerMasker('custom', (value, options) => {
      // Your masking logic
    });
  }
}
