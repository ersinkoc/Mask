FAQ

Frequently asked questions:

How do I mask nested objects?

Use dot notation in the fields option:

mask.mask(data, {
  fields: {
    'user.email': { strategy: 'middle' },
    'billing.card': { strategy: 'last:4' }
  }
});

Can I use custom masking characters?

Yes! Use the char option:

mask.email('test@example.com', { char: 'â€¢' });

How do I validate input before masking?

All plugins include built-in validation. Invalid data will throw an error:

try {
  const masked = mask.email('invalid-email');
} catch (error) {
  if (error instanceof InvalidValueError) {
    console.error('Invalid email');
  }
}

Does @oxog/mask work in the browser?

Yes! It's a pure TypeScript/JavaScript library with no dependencies.

How do I create a custom plugin?

Implement the Plugin interface and register it with mask.use():

const plugin = {
  name: 'custom',
  version: '1.0.0',
  install(kernel) {
    kernel.registerMasker('custom', (value, options) => {
      // Your logic
    });
  }
};
mask.use(plugin);
